UI 布局方案

如何展示全流水线的流式内容？

A

单栏时间线

所有事件按时间顺序流式呈现，像聊天界面。简洁直观，适合观察对话流程。

B

双栏布局

左侧时间线流，右侧显示当前阶段详情（角色统计、评分、文件列表等）。信息更丰富。

C

分阶段标签页

顶部标签切换：竞技场 / 设计 / 构建 / 审查 / 部署。每个阶段独立展示，互不干扰。
